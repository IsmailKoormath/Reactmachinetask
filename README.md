# Reactmachinetask
Statboard


### Backend

1. Navigate to the server directory:
    ## cd server
   
2. Install backend dependencies:
    ## npm install

3. Run the backend server:
    ## node server.js

### Frontend

1. Navigate to the client directory:
    ## cd client
  
2. Install frontend dependencies:
    ## npm install

3. Run the frontend application:
    ## npm start
